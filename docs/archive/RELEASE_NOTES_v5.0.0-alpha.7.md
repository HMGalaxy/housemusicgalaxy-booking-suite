# Galaxy Cue v5.0.0-alpha.7

## Overnight Timeline Engine

- Week and Day calendars now display a continuous timeline from 12:00 AM through 6:00 AM the following day.
- Events that end after midnight remain one continuous block rather than appearing to end before they start.
- Setup and live-event periods are shown as separate timeline phases.
- Next-day hours are labeled with `(+1)`.
- Existing 15-minute scheduling increments remain in place.
- Event-type color classes continue across calendar views.
