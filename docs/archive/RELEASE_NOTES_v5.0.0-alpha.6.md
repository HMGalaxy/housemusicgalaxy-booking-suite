# Galaxy Cue v5.0.0-alpha.7

## Required scheduling data
Every event now requires four operational times:

- Setup start
- Setup end
- Event start
- Event end

The requirement applies to the quick-create/edit event form, all consultation forms, and all event planner forms. Every time field remains restricted to 15-minute increments.

These values are stored with the event workspace so they are available for calendar scheduling, quotes, invoicing, staffing, and future conflict detection.
