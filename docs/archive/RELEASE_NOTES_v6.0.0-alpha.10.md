# Galaxy Cue v6.0.0-alpha.10 — Event Action Center & Document Copies

## Implemented
- Client Event Form is an explicit primary action until actually submitted.
- Prefilled booking-request information no longer counts as a submitted Event Form.
- Submitted Event Forms are stored as permanent event documents.
- Client can view Booking Request and submitted Event Form copies.
- Business Events screen can review the submitted Client Event Form and original Booking Request.
- Print / Save PDF is available for both sides.
- Workflow advances to Quote Preparation only after a real client submission.
