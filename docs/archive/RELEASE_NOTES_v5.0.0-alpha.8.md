# Galaxy Cue v5.0.0-alpha.8

## Timeline Alignment & Breakdown Scheduling

- Corrected the Week and Day timeline to use one consistent 64 px hour scale from midnight through 6:00 AM the next day.
- Removed horizontal timeline scrolling; the schedule now scrolls vertically only and remains aligned with weekday headings.
- Corrected event block positioning and duration rendering.
- Added side-by-side lanes for overlapping event phases.
- Removed Setup End. Setup now runs from Setup Start until Event Start.
- Added required Breakdown End after Event End.
- Added separate Setup, Event, and Breakdown blocks to Week and Day views.
- Preserved 15-minute increments across all scheduling fields.
