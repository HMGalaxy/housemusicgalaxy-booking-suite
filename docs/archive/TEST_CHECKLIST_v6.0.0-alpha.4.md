# Test Checklist — v6.0.0-alpha.4

1. Sign into the Business Portal and copy the Public Booking Link.
2. Open it in a private/incognito window.
3. Confirm the booking request uses four steps: Contact, Event, Services, Details.
4. Confirm Continue does not advance when a required field is empty.
5. Submit a request and confirm the success reference appears.
6. Refresh the Business Dashboard and confirm the request appears under New Booking Requests.
7. Click Review Request and verify all submitted information appears.
8. Test Need More Info and confirm the note and Waiting for Client status appear.
9. Submit a second request, review it, and choose Decline. Confirm it leaves the pending list.
10. Submit a third request and choose Accept & Create Event. Confirm exactly one Client and one Event are created and the event workspace opens.
11. Refresh the application and confirm the accepted Event and Client persist.
12. Confirm the version badge reads 6.0.0-alpha.4.
