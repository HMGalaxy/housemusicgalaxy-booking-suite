# Galaxy Cue v4.2.0 QA Checklist

1. Sign in and confirm the Dashboard shows current cloud event/client counts.
2. Create a client with an email address.
3. Try creating the same client again with the same email; the existing profile should open.
4. Create an event using that email; no duplicate client should be created.
5. Open Events and search by:
   - client name
   - venue
   - date
   - event reference
6. Test all event status filters.
7. Reopen an event consultation.
8. Open the linked client from the event preview.
9. Confirm the client profile shows:
   - total events
   - next event
   - last event
   - linked-event timeline
10. Check Dashboard, Events and Clients on mobile.
