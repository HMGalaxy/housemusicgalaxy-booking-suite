# Galaxy Cue v6.0.0-alpha.8

## Event Activation & Universal Client Portal

- Client Portal is now a single universal magic-link login.
- Removed booking-request tabs and Organization Booking ID from client login.
- Accepting a booking request moves the event workflow to `EVENT_FORM_PENDING`.
- Accepted events receive a client Event Booking Form assignment.
- Client login finds events through the verified client email.
- Client can save a draft or submit the Event Booking Form.
- Submission updates the existing event workspace and advances workflow to `QUOTE_PREPARATION`.
- Business version badge updated to 6.0.0-alpha.8.
