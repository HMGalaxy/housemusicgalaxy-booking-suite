# Test checklist
1. Run `supabase/galaxy_cue_v6_alpha11_workflow.sql`.
2. Accept a booking request and log in as the client.
3. Open the Full Event Workbook and verify the large business-profile form appears.
4. Save a draft, reopen it, then submit.
5. Verify the business event shows Review Full Event Workbook and Create / Send Quote.
6. In Client Portal, use Book a New Event and submit a second event.
7. Verify the new request appears in Leads.
8. Verify Events still has + New Event for in-person/phone requests.
