# Galaxy Cue v5.0.0-alpha.5

## Timeline Calendar & 15-Minute Scheduling

- Week and Day views now display a 6:00 AM–12:00 AM hour grid.
- Events are positioned by start time and sized by their duration.
- Events without an end time default to a two-hour display block.
- All HTML time fields use 15-minute increments.
- Manually typed times snap to the nearest 15 minutes when changed.
- Start and end times persist to local storage and Supabase.
