# Galaxy Cue v5.0.0-alpha.14

## Smart Themes

- Added Auto, Light and Dark appearance modes.
- Auto mode uses the device local time: Light from 7:00 AM to 7:00 PM and Dark from 7:00 PM to 7:00 AM.
- Theme switches without reloading and updates browser/PWA theme color.
- Preference saves locally when signed out.
- Preference and business settings sync through `business_settings` when signed in.
- Added smooth transitions and reduced-motion support.
- Styling scope is now considered complete so development can shift to workflow and connections.
