# Galaxy Cue v5.0.0-alpha.2

## Event editing and deletion fix

- Added an **Edit Event Details** action to the event preview.
- Existing event details now load into the event modal and save back to the same event reference.
- Editing preserves the existing consultation, quote, contract, planner and workflow data.
- Added a confirmed **Delete Event** action.
- Deletion works in both local mode and Supabase cloud mode.
- Event lists refresh immediately after editing or deletion.
