# Deployment

1. Extract `GALAXY_CUE_v7.0.3.zip`.
2. Open the `GALAXY_CUE_v7.0.3` folder.
3. Upload everything inside it to the GitHub repository root.
4. GitHub Settings → Pages → Deploy from branch → `main` → `/ (root)`.
5. Wait for deployment and hard refresh.

URLs:
- Business OS: repository root or `/apps/business/`
- Admin OS: `/apps/admin/`
- Client App: `/apps/client/`
- Platform launcher: `/platform.html`
